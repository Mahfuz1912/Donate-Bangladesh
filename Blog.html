Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for representing HTML, XML, and other structured documents as a tree of nodes.

How do you select an element from the DOM?

To select an element from the DOM, We can use the following methods:
1.getElementById: Selects an element by its ID.
2.getElementsByClassName: Selects all elements with a specific class name.
3.getElementsByTagName: Selects all elements with a specific tag name.
4.querySelector: Selects the first element that matches a CSS selector.
5.querySelectorAll: Selects all elements that match a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach an event listener to a parent element instead of directly to child elements. This improves performance and makes your code more reusable.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, use getAttribute(), setAttribute(), and removeAttribute(). To manipulate styles, use style property and getComputedStyle(), or use CSS classes.
